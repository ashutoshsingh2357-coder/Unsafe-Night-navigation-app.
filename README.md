# Unsafe Night Navigation: AI-Powered Safety-First Navigation

### 🔴 Problem Statement
Night travel is a major risk for millions. Current navigation apps only care about speed and ignore the dangers of the road.
We are all driving into the dark, unaware of construction, poor lighting, or bad weather—leading to accidents every single day.
This is a global crisis that we cannot ignore anymore.

### 🟢 Solution
We created **Unsafe Night Navigation**. We are changing how the world navigates by making safety the top priority.
Our AI engine analyzes real-world conditions to provide a **Safety Score** for every route, so we choose the safest path, 
not just the fastest, before we even start our engine.

### 🔢 The Safety Score System
We assign a **Dynamic Safety Score (0–100)** to every route, calculated in real-time based on:
* **Infrastructure Risk:** Road lighting levels and active construction/blockages.
* **Environmental Hazards:** Rain impact and visibility risks (fog, smog, and air quality).

### 🚀 Key Features
* **Safety-First Routing:** Automatically suggests routes with a Safety Score of 80+, prioritizing well-lit and secure paths.
* **Turn-by-Turn Navigation:** Real-time map guidance that updates dynamically based on road conditions.
* **AR Mode:** An AR-powered mode that provides better situational awareness and road clarity in low-light environments.
* **Integrated Safety Toolkit:**
    * **SOS Alert:** One-tap emergency notification to trusted contacts and the nearest police station.
    * **Fake Call:** A simulated incoming call system to deter unwanted attention.
    * **Hazard Reporting:** Real-time crowd-sourced reporting of potholes, blockages, or dark zones.
    * **Voice Control:** Hands-free operation using simple voice commands like "Unsafe Night Navigation, help."
